# 🛡️ Sentinel AI: Autonomous Cybersecurity Analyst

Sentinel AI is a Deep Learning-based intrusion detection system that transforms network logs into 8x8 heatmaps for pattern recognition.

## 🚀 Features
- **Feature 1:** CNN-based image classification for threats.
- **Feature 2:** RAG-based vulnerability remediation.
- **Feature 3:** Privacy Guardrails for IP redaction.
- **Feature 4:** Interactive Gradio Dashboard.

## 🛠️ Setup
1. Install requirements: `pip install tensorflow pandas gradio matplotlib`
2. Run Dashboard: `python sentinel_dashboard.py`
